# Network Availability and Latency Monitoring Tool

This project is a simple network monitoring tool built with Python to track device availability, latency, and packet loss.

## Objective

To improve visibility into network behavior by monitoring device reachability and logging basic performance data.

## Features

* Checks device reachability
* Measures latency
* Detects packet loss
* Logs results

## Files

* monitor.py
* devices.txt

## Skills Demonstrated

* Network monitoring
* Troubleshooting support
* Basic automation
